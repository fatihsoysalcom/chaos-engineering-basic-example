# Chaos Engineering Basic Example

This Python script simulates a system processing user requests. It demonstrates chaos engineering principles by introducing random failures into user actions, highlighting that errors are a normal part of complex systems.

## Language

`python`

## How to Run

Save the code as `chaos_demo.py` and run it from your terminal using: `python chaos_demo.py`
